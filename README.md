# ダンジョン探索 (Unity)

Unity（C#）で制作した2Dローグライクゲームです。  
ダンジョン自動生成・階層遷移・セーブ/ロードなど、ゲームシステムのコア部分を中心に個人開発しました。

---

##  主な機能

- ダンジョン自動生成（Cellular Automata）
- 3階層構成（1F⇔2F⇔3F）
- 階段座標の階層間固定システム
- 宝箱レアリティ抽選
- アイテム管理 / ロードアウト
- セーブ / ロード（ラン途中再開）
- Tilemapベース2Dマップ

---

##  技術スタック

- Unity
- C#
- Tilemap
- ScriptableObject
- JsonUtility（セーブデータ管理）
- Git / GitHub

---

##  工夫した点

- 階段座標をフロア間で共有し、整合性のある階層遷移を実現
- ラン状態をJSON保存して再現性を担保
- データ駆動設計により拡張しやすい構造に設計
- 生成ロジック・進行管理・UIまで全て個人実装

---

##  開発目的

ゲーム制作を通して、  
- アルゴリズム実装力  
- 設計力  
- データ管理  
を高めることを目的に制作しました。
